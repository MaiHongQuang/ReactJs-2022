git init: Tạo version git ở phía local: => đối chiếu với repo

git ađd . : Cho tất cả các file/folder vào git local

git status : Kiểm tra xem các file local đã có trong git local chưa

git commit -m "first commit" : Tất cả các file được add vào git local sẽ ném lên git remote

git remote add origin https://github.com/MaiHongQuang/ReactJs-2022.git : thêm link remote vào local
